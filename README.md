## üìñ Project Description

**Conco-ordinates** is a campus navigation and guidance web application designed to help Concordia University students navigate efficiently across both SGW and Loyola campuses. The system provides outdoor and indoor directions, integrates academic schedules, and supports accessibility-aware routing. By combining interactive maps, real-time navigation, and institutional data, the application aims to improve students‚Äô daily campus experience.

---

## üéØ Project Objectives

- Explore detailed maps of SGW and Loyola campuses  
- Provide outdoor navigation between buildings  
- Support indoor navigation within buildings and across floors  
- Generate directions to a student‚Äôs next class based on time and location  
- Highlight indoor and outdoor points of interest  
- Support accessibility-friendly routing

## üõ† Languages & Technologies

- **Platform:** Native Android (Mobile)
- **Primary Language:** Kotlin
- **Frontend/UI:** XML Layouts, Material Design 3 (M3)
- **Backend/Logic:** Android Jetpack, FusedLocationProviderClient
- **APIs & Services:** - **Google Maps SDK for Android** (Map rendering & Camera control)
  - **Google Maps Utility Library** (PolyUtil for geofencing & distance logic)
  - **Google Calendar API** (Class schedule integration)
  - **Concordia Open Data API** (Shuttle & Building data)
- **Architecture:** Model-View-Controller (MVC) / MVVM
- **Methodology:** Agile / Scrum
- **Version Control:** GitHub
